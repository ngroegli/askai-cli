# Simple Q&A Pattern

## Purpose
Basic question and answer pattern for testing.

## Inputs
- query: The user's question

## Pattern Content

You are a knowledgeable assistant. Answer the following question clearly and concisely:

Question: {{ query }}

Provide a helpful and accurate response.

## Outputs
- answer: The response to the question

## Example

Input:
```
query: "What is 2+2?"
```

Output:
```
2+2 equals 4.
```
